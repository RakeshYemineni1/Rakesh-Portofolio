# Portfolio Resume Update Instructions

## How to Update Your Resume

1. Save your updated resume as `Rakesh_Resume.pdf`
2. Place it in the `resume` folder (replace the existing file)
3. The portfolio will automatically link to the new resume

## Current Resume Location
`portfolio/resume/Rakesh_Resume.pdf`

## Note
- Keep the filename as `Rakesh_Resume.pdf` for the download link to work
- Or update the link in `index.html` line with: `href="resume/YOUR_NEW_FILENAME.pdf"`
